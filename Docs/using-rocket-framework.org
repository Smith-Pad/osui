#+TITLE: Using Rocket Framework 

* Introduction: 

These notes are used for experimenting with the Rocket Framework on Rust 


* Steps for creating rendered html

** Step 1: Create a Rocket.toml file next to the Cargo.toml file

The ~paths = ["static"] is where you include the CSS and HTML files to 

#+BEGIN_SRC c
[static]
paths = ["static"]

#+END_SRC



** Step 2: Create a Rust Program 

This is where it will allow the ability to display the UI. 

#+BEGIN_SRC c

#[macro_use]
extern crate rocket;

use rocket::fs::NamedFile;
use std::io;

#[get("/")]
fn index() -> io::Result<NamedFile> {
    NamedFile::open("static/index.html")
}

#[launch]
fn rocket() -> _ {
    rocket::build().mount("/", routes![index])
}
#+END_SRC


* Step 3: Run the program

#+BEGIN_SRC shell
    cargo build
    cargo run
#+END_SRC
